# Invoice Signature Tracker

Open `index.html` in a browser to use the private invoice signature tracker built from `D:\Downloads\Invoice Master.xlsx`.

## What is included

- Login access from the workbook `Users` sheet
- Assam admin access to all labs, districts, and centers
- Lab admin access restricted to the assigned lab only
- Dashboard metrics for the current user scope
- Lab, district, status, and center-name filters
- Center workflow for Active/Inactive, then RG, Hospital, DPM, and Joint Director
- Browser localStorage persistence so each center resumes from the last saved status

## Example logins

- Assam admin: `assam@admin.com` / `Assam@11`
- Kamrup admin: `kamrup@admin.com` / `Kamrup@11`

This is a static private site prototype. For multi-user production use, connect the tracker to a backend database and server-side authentication.
